
# Projeto Motorhome — FIAT Doblò 2015

## 1. Visão geral
- **Veículo:** FIAT Doblò 2015 (versão passageiro)
- **Conceito:** mini motorhome modular, mantendo uso urbano no dia a dia e convertendo para viagem
- **Modo de uso:**
  - **Modo cidade:** bancos traseiros + caixote/sofá no lugar do porta-malas
  - **Modo viagem:** retira apenas os bancos traseiros; o caixote permanece e serve como base estrutural da cama (casal apertado)

## 2. Layout — lateral direita (lado motorista)
Sentido frente → traseira:
1. Banco do motorista (original)
2. Armário alto (pode avançar à frente da janela; altura máxima que o teto permitir)
3. Bancada pequena para fogareiro
4. Cuba de pia pequena

## 3. Layout — lateral esquerda (lado passageiro)
Sentido frente → traseira:
1. Banco do passageiro (original)
2. Segunda fileira — banco traseiro bipartido (mantido, removível para modo cama)
3. Caixote/sofá no lugar do porta-malas (com baú interno para storage e servindo como base da cama)

## 4. Corredor central / modo cama
- Corredor entre as duas laterais na parte traseira
- **Conversão p/ viagem:** remove apenas os bancos traseiros bipartidos; o caixote NÃO sai — é a base estrutural da cama
- Cama final: casal apertado (estrado se apoia sobre o caixote + extensão sobre o corredor / vão dos bancos removidos)
- Definir: sistema de extensão do estrado (tábuas removíveis? articulado?) e onde guardar as peças quando em modo cidade

## 5. Sistemas

### 5.1 Água
- Reservatórios externos sob o assoalho (limpa + cinza), maior capacidade que o espaço permitir
- Bombas **elétricas** 12V (pressostato recomendado)
- Torneira/filtro na cuba

### 5.2 Energia
- **Bateria auxiliar** (estacionária / deep cycle) isolada da bateria de partida
- **Placa solar** no teto
- **Carga via alternador** com o motor ligado (DC-DC charger recomendado para preservar a bateria)
- **Entrada para tomada comum (shore power)** com carregador
- **Inversor 110V** para tomadas internas
- Quadro de distribuição 12V para iluminação, bombas e exaustor

### 5.3 Gás / cocção
- Fogareiro principal: **cartucho descartável**
- Adaptação prevista para **botijão P2 / P5** (engate alternativo + ventilação adequada)

### 5.4 Ventilação
- **Tela mosquiteira** em uma das janelas laterais
- **Exaustor no teto** 12V (idealmente reversível: entrada e saída de ar)

### 5.5 Iluminação interna
- Luminárias **LED 12V** distribuídas: cozinha, sofá/cama, teto

## 6. Altura interna
- Doblò tem pé-direito baixo — armários altos limitados pela altura do teto
- Priorizar armários sobre a área da cozinha (lateral direita), liberando circulação no corredor
